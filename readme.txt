A lightweight react based ajax live search plugin, use [bestwp-ajax-search] shortcode to show the search box anywhere you want.

== Description ==
Best wordpress ajax search plugin will help you to get rid of the core WordPress search plugin.
to active the search box you need to add the "[bestwp-ajax-search]" shortcode where your want to show the ajax search box.
 
**KEY FEATURES**
* React base lightweight plugin
* Search in posts and pages
* Live search (Ajax serach)

== Installation ==
You can install Best WP Ajax Search Downloads by either using wordpress.org plugin directory, or by directly uploading plugin files to your server.

== Frequently Asked Questions ==
= How can I use this plugin?
It's really easy just add this shortcode [bestwp-ajax-search] to a page of your website and make sure the best wp ajax search plugin is active.
